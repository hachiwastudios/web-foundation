## Usage

### Class Autocomplete
Type `class="` in HTML and get suggestions:
```html
<div class="flex gap-4 p-8">
```

### Icon Autocomplete
Type `data-icon="` and get icon suggestions:
```html
<i data-icon="heart" class="icon"></i>
```

### Snippets
Type `wf-` to see all available snippets:
- `wf-container` - Responsive container
- `wf-card` - Card component
- `wf-button` - Button
- `wf-icon` - Icon
- `wf-grid` - Responsive grid
- `wf-flex` - Flex container
- `wf-navbar` - Navigation bar

## Installation

### From Marketplace
1. Open VSCode
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Web Foundation"
4. Click Install

### Manual Installation
1. Download the `.vsix` file
2. Open VSCode
3. Go to Extensions
4. Click "..." ‚Üí "Install from VSIX"
5. Select the downloaded file
